# ピクセルピコ カラーマッチャー

アイロンビーズショップ「[ピクセルピコ](https://www.pixelpico.com/)」で取り扱われているビーズの色から、画像の任意の位置の色に最も近い色を3候補提案するブラウザツールです。

## 使い方

1. ページを開いて画像を選択
2. 拡大表示された画像の好きな位置をクリック
3. クリック位置のRGBに最も近いビーズ色TOP3が右側に表示されます

各候補にはA(フューズビーズミニ 2.6mm)、R(フューズビーズミドル 5mm)、PN(ピクセルビーズミドル 5mm)の品番が表示されます。

## ローカルでの動作確認

`fetch()` を使っているため `file://` で直接開くと動きません。簡易サーバーを立ててください。

```bash
# Python 3
python -m http.server 8000
# その後 http://localhost:8000 にアクセス
```

## デプロイ

Cloudflare Pages にGitHubリポジトリを接続して自動デプロイしています。
mainブランチへのpushで自動的に反映されます。

## データソース

ビーズの色情報は [ピクセルピコ公式RGBカラー対応表](https://www.pixelpico.com/page/6) を元にしています。
